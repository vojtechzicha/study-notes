Numerické metody 1

Korektnost úloh a stabilita výpočetních postupů

Definice: Korektní úloha na dvojici \left( B_{1},B_{2} \right) je taková úloha y = U(x),x \in B_{1},y \in B_{2}, kde

  • \forall x \in B_{1}:\exists!y \in B_{2};

  • x_{n} \rightarrow x,U\left( x_{n} \right) = y_{n} \Rightarrow U\left( x_{n} \right) \rightarrow U(x) = y, tj. řešení y spojitě závisí na x.

Definice: Dobře podmíněná úloha u(x) je taková úloha, kde malá změna x vyvolá malou změnu y. Pro dobře podmíněnou úlohu platí1 < c_{p} < 100, kde:

c_{p} = \frac{\frac{\left\| \Delta y \right\|}{\left\| y \right\|}}{\frac{\left\| \Delta x \right\|}{\left\| x \right\|}}

Aproximace čísla

Nechť x je přesná hodnota a \widetilde{x} \approx x je aproximace hodnoty. Absolutní chyba vyjadřuje \Delta x = x - \varepsilon, přičemž hledáme odhad absolutní chyby \left\| \Delta x \right\| \leq \varepsilon. Relativní chyba vyjadřuje \frac{\Delta x}{x}, přičemž hledáme odhad relativní chyby \frac{\left\| \Delta x \right\|}{\left\| x \right\|} = \frac{\left\| \Delta x \right\|}{\left\| \widetilde{x} \right\|} \leq \delta.

Metoda půlení intervalů

Mějme a_{0},b_{0} takové, že f\left( a_{0} \right) \cdot f\left( b_{0} \right) < 0. Konstruujme posloupnosti \left( a_{k} \right),\left( b_{k} \right) takových, že \left\langle a_{0},b_{0} \right\rangle \supset \left\langle a_{1},b_{1} \right\rangle \supset \ldots \supset \left\langle a_{k},b_{k} \right\rangle \supset \ldots a současně \forall i \in \mathbb{N}^{0}:f\left( a_{i} \right) \cdot f\left( b_{i} \right) < 0. Máme-li \left\langle a_{k},b_{k} \right\rangle, vypočteme s_{k + 1} = \frac{a_{k} + b_{k}}{2}. Je-li f\left( a_{k} \right) \cdot f\left( s_{k + 1} \right) < 0, poté a_{k + 1} ≔ a_{k} a b_{k + 1} ≔ s_{k + 1}. Je-li f\left( b_{k} \right) \cdot f\left( s_{k + 1} \right) < 0, poté b_{k + 1} ≔ b_{k} a a_{k + 1} ≔ s_{k + 1}.

Věta (konvergence půlení intervalů): Metoda půlení intervalů je konvergentní a navíc \lim_{k \rightarrow \infty}s_{k} = \overline{x}.

Nechť \overline{x} \in \left\langle a_{k},b_{k} \right\rangle,s_{k + 1} = \frac{a_{k} + b_{k}}{2},d_{k} = \frac{\left( b_{k} - a_{k} \right)}{2}. Víme, že a_{k} = s_{k + 1} - d_{k},b_{k} = s_{k + 1} + d_{k},\overline{x} \in \left\langle s_{k + 1} - d_{k},s_{k + 1} + d_{k} \right\rangle. Tudíž \left| \overline{x} - s_{k + 1} \right| \leq d_{k}. Dále b_{k} - a_{k} = \frac{b_{k - 1} - a_{k - 1}}{2} = \ldots = \frac{b_{0} - a_{0}}{2^{k}} a 0 \leq \left| \overline{x} - s_{k + 1} \right| \leq d_{k} = \frac{b_{0} - a_{0}}{2^{k - 1}} \rightarrow 0. Víme tudíž, že \lim_{k \rightarrow \infty}\left| \overline{x} - s_{k + 1} \right| = 0 \Rightarrow \lim_{k \rightarrow \infty}s_{k + 1} = \overline{x}.\blacksquare

Metoda prostých iterací

Metoda prostých iterací je založena na konstrukci posloupnosti aproximací \left( x_{k} \right),k \in \mathbb{N}^{0} daných rekurzivním předpisem x_{k + 1} = G\left( x_{k} \right).

Definice: Zobrazení G:\mathbb{R}^{n} \rightarrow \mathbb{R}^{n} se nazývá kontraktivní, platí-li:

\exists\varrho \in (0,1):\forall x,y \in \Omega:\left\| G(x) - G(y) \right\| \leq \varrho \cdot \left\| x - y \right\|

Definice: Nechť je zobrazení G:\mathbb{R}^{n} \rightarrow \mathbb{R}^{n}. Bod \overline{x} \in \mathbb{R}^{n} se nazývá pevný bod zobrazení G, platí-li \overline{x} = G\left( \overline{x} \right).

Věta (pevný bod): Nechť \Omega \subset \mathbb{R}^{n} je souvislá, omezená a uzavřená množina, zobrazení G:\mathbb{R}^{n} \rightarrow \mathbb{R}^{n} je kontraktivní v \Omega a postupně aproximace x_{k} : = G\left( x_{k - 1} \right) leží v \Omega, poté:

  • \forall x_{0} \in \mathbb{R}^{n} platí, že posloupnost \left( x_{k} \right) konverguje a \lim_{k \rightarrow \infty}x_{k} = \overline{x};

  • \overline{x} je pevným bodem G;

  • \left\| x_{k} - \overline{x} \right\| \leq \frac{\varrho^{k}}{1 - \varrho} \cdot \left\| x_{1} - x_{0} \right\|

Věta (postačující podmínka konvergence metody prostých iterací): Nechť \Omega \subset \mathbb{R}^{n} je souvislá, omezená a uzavřená množina, zobrazení G:\mathbb{R}^{n} \rightarrow \mathbb{R}^{n} je kontraktivní v \Omega a postupně aproximace x_{k} : = G\left( x_{k - 1} \right) leží v \Omega, nechť G \in C^{2}(\Omega) a \forall x \in \Omega:\left\| G'(x) \right\| \leq M < 1, poté platí tvrzení věty o pevném bodě.
